Concept of Limits

The formal definition of a limit is a mathematical concept used to describe the behavior of a function as it approaches a specific value. In calculus, the limit of a function is denoted by the symbol lim and is expressed as the value that the function approaches as the input variable approaches a certain value. The formal definition of a limit states that if a function f(x) approaches a limit L as x approaches a value a, then for every positive number ε, there exists a positive number δ such that if 0 < |x - a| < δ, then |f(x) - L| < ε. In simpler terms, this means that as the input variable gets closer and closer to the value a, the function f(x) gets closer and closer to the limit L.

Examples of limits include:

  • The limit of f(x) = 1/x as x approaches infinity is zero.
  • The limit of f(x) = sin(x)/x as x approaches zero is one.
  • The limit of f(x) = 1/x as x approaches zero does not exist.

The objectives of this lesson are for students to:

  • Understand the formal definition of a limit.
  • Compute limits using algebraic and graphical methods.
  • Identify and interpret the behavior of functions as they approach a limit.
  • Apply the concept of limits to solve real-world problems.

The formal definition of a limit can be expressed using mathematical notation and vocabulary as follows:

Let f(x) be a function defined on an interval containing a, except possibly at a itself. We say that the limit of f(x) as x approaches a is L, denoted by

lim x→a f(x) = L,

if for every ε > 0, there exists a δ > 0 such that if 0 < |x - a| < δ, then |f(x) - L| < ε.

In this definition, ε and δ are positive numbers that represent the size of the error allowed between the function value and the limit. If we can find a small enough δ for any ε, then we say the limit exists and is equal to L. This definition captures the idea that the function approaches the limit as we get closer and closer to the point a, but it may not necessarily be defined at a.

To illustrate the concept of a limit graphically, consider the function f(x) = (x^2 - 1)/(x - 1). If we plot this function, we see that it is undefined at x = 1 because of the division by zero.
